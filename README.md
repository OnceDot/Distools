> Light and powerful utility to enhance your Discord experience.


### Features


- Delete messages
    - [ ] Everywhere
    - [X] Guild
    - [X] Channel
    - [X] DM
- Replay messages
    - [ ] Deleted
    - [ ] Modifications
- Backup messages
    - [ ] Everywhere
    - [ ] Guild
    - [ ] Channel
    - [ ] DM
- Backup guilds
    - [ ] Ranks
    - [ ] channels
    - [ ] messages
